# Pilates Fountain Hills

An exclusive boutique reformer Pilates studio website — Fountain Hills, Arizona.

## What's here

A single-page, dependency-free static site:

- `index.html` — full marketing page (hero, studio, classes, schedule, memberships, booking, testimonials, FAQ, contact)
- `styles.css` — design system (cream / sage / terracotta palette, Cormorant Garamond + Inter)
- `script.js` — interactive booking calendar with day-of-week class lineups and time-slot selection

## Highlights

- **15 reformers**, classes capped to match
- **8 classes daily** Monday–Friday; 4–6 on weekends (Sat 6 / Sun 4)
- **Unlimited Membership — $289/month** with custom welcome package and monthly grip socks
- **Single sessions $35** when openings exist
- **Private 1:1** sessions
- Live booking calendar with availability indicators (available / limited / full)
